How to Check Live Firestore Occupancy:

  1. Go to Firebase Console (console.firebase.google.com) ➔ Select Project WalletVibe.
  2. Click Firestore Database in the sidebar ➔ Select the Usage tab at the top.
  3. Inspect Stored Data (shows exact Storage size in Bytes / MB / GB).

What happens if Storage reaches 90% (900 MB)?

  • Automatic Purge / Archive Protocol: Implement snapshot archival for transactions older than 180 days to keep active document count ultra-lean.
  • Blaze Pay-As-You-Go Upgrade: Upgrading to Blaze removes the 1 GB wall. Storage above 1 GB costs only $0.18 per GB per month (approx ₹15/month for an extra GB).
  • Local Cache Optimization: WalletVibe already leverages local snapshot caching (wv_cached_snapshot_*) so clients only sync changed delta documents.
